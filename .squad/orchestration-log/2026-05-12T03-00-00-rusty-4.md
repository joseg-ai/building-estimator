## Key Technical Fixes

1. **uuid v13→v9** — uuid v13+ is ESM-only. Downgraded to v9.0.1 for CJS `require()` compatibility.

2. **ubuntu-22.04 runner** — Azure glibc 2.36 requires compiled binaries from 2.35 or lower. ubuntu-24.04 produces 2.38 (crash).

3. **OIDC federated identity** — Azure MCAP policy redacts basic-auth Kudu credentials. OIDC avoids hardcoded secrets.

4. **mkdirSync in db.js** — DB directory creation moved from `index.js` to `db.js` to ensure dir exists before route imports trigger `new Database()`.
