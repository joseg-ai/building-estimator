# Session Log — Azure Login Fix 2026-05-12T17:43:33

**Focus:** Resolve "Failed to fetch" on Azure production login

## Problem

Azure App Service (HTTPS) was blocked from calling `http://localhost:3001/api` due to mixed-content security policy.

## Solution Summary

- Fixed hardcoded localhost URL in `webapp/src/api.ts` to use env-based `VITE_API_URL ?? '/api'`
- Set `VITE_API_URL='/api'` in deploy workflow (not empty string)
- Leveraged Vite proxy for local dev (no CORS needed)

## Result

App deployed successfully; login now works on Azure.
